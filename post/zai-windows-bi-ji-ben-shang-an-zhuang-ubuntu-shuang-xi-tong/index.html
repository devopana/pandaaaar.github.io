在Windows笔记本上安装Ubuntu双系统

2024-03-06 5 min read

老师留了装Linux虚拟机的作业，正好我要把双系统装回来，还有人问我装双系统的事，所以就直接写一个教程出来了

准备材料：

  • Windows笔记本（设备正在充电或者电量充足）
  • 8G以上的U盘
  • Rufus：官网
  • Ubuntu桌面版镜像：下载地址
  • （非必需）DiskGenius（或者系统自带的磁盘管理）

一、准备工作

1. 检查磁盘剩余空间

打开DIskGenius或磁盘管理（此电脑右键——显示更多选项——管理——磁盘管理），确认你的硬盘里有足够的未分配空间留给Ubuntu（下图以DiskGenius为例）

分配空间因工作强度而异，个人建议不少于60G

2. 下载Ubuntu镜像

去Ubuntu官网下载桌面版镜像，我这里为了图新鲜选择了最新的23.10版本，追求稳定可以选择LTS的22.04版（校园网非包月同学注意，桌面版镜像要4-5个G，注意流量开销）

下载后是如图所示的.iso镜像文件

3. 制作启动盘

插入准备好的U盘，打开Rufus（下载链接），在设备栏正确选择你刚刚插入的U盘，不确定可以通过名称、容量判断是否为刚刚插入的U盘（一定要选对！因为制作时会清除U盘所有数据！选错后果自负！！！）
引导类型选择镜像文件，点击右面的选择，在弹出界面里正确选择你刚刚下载的Ubuntu桌面版镜像
其余配置按下图配置就好

确认无误后点击下面的开始，会弹出如下提示

选择默认选项，即以ISO镜像模式写入即可
然后等待下面的进度条读完，就代表启动盘制作完成了

二、安装系统

我没有采集卡，所以从这里往下基本都是拿手机拍照的画面了，观感不好请见谅

1. 进入BIOS设置启动顺序

确定U盘是插入状态，然后重启电脑，在启动过程中不断按F2进入BIOS（进入方法因电脑品牌而异，如果F2不行可以去搜索自己品牌对应的进入方法）
进入启动配置界面，把刚刚制作的启动盘调整为第一个启动

点击下方保存设置，会重新启动电脑（如果没有就先保存后退出，然后再次启动）

2. 安装系统

BIOS的启动顺序调整无误的话，重启电脑会进入一个黑屏选择栏，选择它默认的Try and Install Ubuntu就行
稍等片刻就可以看见如下的安装界面
界面语言建议选择英文，因为用户文件夹名的生成是基于界面语言的，英文文件夹可以预防一些问题，后续界面语言还可以调回来（也可以先选中文，同理后续调一下英文，改文件夹名字）

选择Install Ubuntu

键盘布局选择英文

连接网络，有线的同学可以选择第一个Wired connection，但是我这里没有接入网线，所以选择Connect to a Wi-Fi network，选择Wi-Fi名，点击右下角的Connect即可

小小补充下，使用校园网的同学如果没有认证的话，可以连接校园网后，点击左上角小白条，然后打开左侧栏的火狐浏览器，进入认证界面完成认证



可能会遇到热更新的情况，我这里选择更新至最新版（再次提醒流量开销）

选择安装模式，推荐选择Full installation全量安装尝试下
下面的第一个复选框，英伟达显卡用户强烈推荐选择，可以一键安装图形驱动（Linux之父：NVIDIA, fxxk you!）
第二个复选框是附加媒体格式支持，可以选择
选择好后点击下一步（再再次提醒流量开销）

选择Install Ubuntu 23.10.1 alongside Windows Boot Manager，与Windows共存
在这一步也可以手动对分区进行操作，即最后一个选项，建议不了解的不要使用这种方法

下一页会显示它即将要对对应分区进行的更改，如图所示，确认无误点击Install安装

选择时区，这里直接点击地图上的中国板块就可以选择到上海时区了

设置账户名和密码，每次开机登录或者做需要管理员权限的操作时，都会验证此密码，请牢记

选择主题风格，根据个人喜好选择即可，点击Next进行安装

等待安装过程结束，出现如图所示提示，就可以选择Restart now进行重启了

点击重启过后，稍等片刻会出现如图所示提示，此时先拔掉U盘，再按回车键，系统便会重启
电脑屏幕灰尘有点重，不要在意

重启后即可进入系统，开始体验Ubuntu吧😋
